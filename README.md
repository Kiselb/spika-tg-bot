# spika-tg-bot

# Telegram Бот Диагностики Типов Мышления

## Данный Telegrm Бот разрабатывается в рамках стажировки Университета Искусственного Интеллекта по проекту SPIKA

#### Workflow функционирования бота

1. Получение ответов на 5 вопросов профиля
2. AI-Диагностика по 5 вопросам
3. Получение ответов на 38 вопросов
4. AI-Диагностика по 38 вопросам 
5. Определение рекомендованных типов мышления для развития
6. AI-Диагностика ценностей
7. Предоставление итогового отчёта

#### Ограничения

1. Количество символов в ответах на вопросы: 3 - 2048
2. Редактирование отапрвленных ответов на вопросы запрещено
